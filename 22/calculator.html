计算平均数 ​

使用公式 $$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$$

(Markdown)
$$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$$

以下代码输出平均值。

python

T_bar= sum(T) / n
